# Portfolio

This is a basic personal portfolio website.

## Project Files

- `index.html` - the HTML structure
- `styles.css` - the CSS styles
- `app.js` - JavaScript file, empty for now
- `assets/images/` - folder for images

## About This Project

The portfolio includes:

- A header and navigation
- An about section with a short bio and photo
- A projects section with project cards
- A contact section with a form and email
- A footer with copyright and social links
- A Profile photo
